# LSe314
For documenting progress of designing LSe (Light Sequencer), which is a light controlled audio instrument based on RPI, arduino and pure data and which I have started more then a decade ago

Youtube playlist presenting the project: https://youtube.com/playlist?list=PLf1az445lxj2zLXdf2pzVIY8PzWo5s3vG&si=jnik3SH8J_M2ij3X
